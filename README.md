### 기능설명
 - 지출 금액, 부분 관련 내용 그래프화
 - 지출 부분 카테고리화 하여 평균 지출 부분 파악하기
 - 고정 지출 부분 자동으로 작성시키기
 - 목표 금액 설정 후 얼마나 절약, 초과 했는지 파악하기
<br><br/>
