### 사용자의 지출 및 수입 금액을 관리하고, 거래 내역에 대한 리포트를 제공하는 가계부 앱입니다.

<br/>

## 🧐 앱 목적
목적1. 지출 목표 금액을 설정해 사용자의 소비 습관 개선<br>
목적2. 월별 거래 내역에 대한 리포트를 제공해 소비 패턴 정보 제공
