# 아껴쓸랩
이용자의 소비 내역을 확인할 수 있는 앱
<br></br>


## 소개<hr/>
사용자의 편의를 고려한 기능들을 추가한 가계부 작성앱이다.
<br></br>


### 앱목적
지출내역을 기록하고 카테고리 별 그래프를 제공하여 사용자의 소비습관을 개선
<br></br>

### 기능설명
 - 지출 금액, 부분 관련 내용 그래프화
 - 지출 부분 카테고리화 하여 평균 지출 부분 파악하기
 - 고정 지출 부분 자동으로 작성시키기
 - 목표 금액 설정 후 얼마나 절약, 초과 했는지 파악하기
